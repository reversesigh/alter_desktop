### ALTER_desktop  
Create an ALTER desktop application using `nativefier`  
  
## Snap  
`git clone https://github.com/reversesigh/alter_desktop.git`  
`cd alter_desktop`  
`chmod +x snap_install.sh`  
`./snap_install.sh`  
  
## npm  
`git clone https://github.com/reversesigh/alter_desktop.git`  
`cd alter_desktop`  
`chmod +x npm_install.sh`  
`./snap_install.sh`
